  Company Overview                                                                                                                                                               
                                                                                                                                                                                 
  World is building a real human network for the AI age. The company operates the World ID verification system, which includes mobile apps, the Orb hardware device, cloud APIs, 
  and blockchain integration on Ethereum-based Worldchain. Currently, 17 million+ users across 160 countries have verified their identity through this platform, with over       
  350,000 verifications weekly.                                                                                                                                                  
                                                                                                                                                                                 
  About the Team                                                                                                                                                                 
                                                                                                                                                                                 
  The Detection & Response team focuses on protecting the World ID system from attacks. This multidisciplinary group applies expertise across mobile devices, hardware, cloud    
  infrastructure, application security, intrusion detection, machine learning, blockchain, and incident response.                                                                
                                                                                                                                                                                 
  Role Description                                                                                                                                                               
                                                                                                                                                                                 
  This internship involves working on a "distributed analytics system" where the goal is to "execute a smart contract that incentivizes third parties to run specific code       
  against specific datasets, publish outputs, and prove calculations were performed correctly." The project represents blockchain-based verifiable compute applied to detection  
  engineering. It's in early stages, with expectations limited to what's achievable within an internship timeline.                                                               
                                                                                                                                                                                 
  Key Responsibilities                                                                                                                                                           
                                                                                                                                                                                 
  - Contribute to building an automated detection and response system protecting millions of daily identity and financial transactions                                           
  - Help scale protections to billions of future users on trusted and untrusted hardware                                                                                         
  - Work on publishing audit events to public blockchain for transparency                                                                                                        
  - Develop decentralized detection approaches while maintaining privacy
